# what is port securoty

# Descriptio 
Port Security is a technique used to secure network ports by allowing only trusted devices to connect, preventing unauthorized access and protecting the network from 
attacks

# Tools Used
- Cisco Packet Tracer
- Basic Networking Concapts

# What I Did

- Configured port security on switch ports
- Allowed only specific devices to connect
- Restricted unauthorized devices
- Set violation mode to shutdwon

# Steps

1. Assigned devices with specific ip addresses
2. Connected devices to switch ports
3. Configured port security
4. Tested unauthorized access
5. Switch port shutdown when unknown device connected

# Result

Unauthorized devices were blockedand ports were automatically shut down when violation accurred
